Tentang Kami

Podcast Webdev adalah konten podcast yang dinaungi oleh komunitas webdev yang membuat konten podcast seputar web development, seperti podcast dengan web developer, cerita pengalaman web developer, tips web developer, dan juga konten tutorial mengenai web development seperti tutorial CSS, HTML, Javascript, Boostra, Laravel, dan Brain Storming Ide Website.

Komitmen Kami

Komitmen kami adalah membuat konten podcast dan konten video podcast tutorial mengenai web development untuk sharing dan membantu web developer Indonesia agar dapat belajar web development dengan lebih baik sehingga dapat membuat website dengan kualitas yang lebih baik.
